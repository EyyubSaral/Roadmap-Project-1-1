Tabbed Content Example

This is a simple tabbed content interface using HTML, CSS, and JavaScript.

Features

Tabs: Clickable spans at the top that act as navigation for content sections.

Content sections: Each tab displays its corresponding content.

Active tab styling: The currently selected tab changes color.

Hover effect: Tabs highlight when hovered.

Smooth transitions: CSS transitions for hover and content appearance.

Usage

Open the HTML file in a browser.

Click on any tab to display its content.

Only one content section is visible at a time.

Structure

<nav id="tabs"> contains the tab buttons (<span> elements).

<main class="main"> contains the content sections (<div> elements).

CSS handles styling, hover, and transitions.

JavaScript toggles visibility and active tab color.
